# React Contact Form with LocalStorage & React Router

## eskripsi

Project ini adalah aplikasi React sederhana yang memiliki:

- Form kontak untuk input `Name`, `Email`, `Phone`, `Subject`, dan `Message`.
- Menyimpan data form ke **localStorage**, sehingga data tetap tersimpan setelah halaman di-refresh.
- Menampilkan data tersimpan di bawah form.
- Sudah menggunakan **React Router** untuk navigasi antar halaman.

---

## teknologi yang digunakan

- React 
- React Router Dom 
- TailwindCSS
- Vite (sebagai bundler & dev server)
